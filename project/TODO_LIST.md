# 项目开发计划 (Todo List)

本计划基于 `系统分析设计文档.md`，分为后端、后台管理、移动端三个主要部分。

## Phase 1: 后端开发 (Node.js + Express + MySQL)
- [ ] **1.1 项目初始化**
    - [ ] 创建 `be` 目录，初始化 `package.json`。
    - [ ] 安装依赖 (`express`, `mysql2`, `sequelize`, `cors`, `jsonwebtoken`, `dotenv`, `morgan`).
    - [ ] 配置 `app.js` 基础服务 (CORS, BodyParser, Static).
- [ ] **1.2 数据库搭建**
    - [ ] 编写 SQL 初始化脚本 (`db/init.sql`)。
    - [ ] 创建数据库连接配置 (`config/database.js`)。
    - [ ] 定义 Sequelize 模型 (`User`, `Game`, `Order`, `CompanionService`, `WalletLog`).
- [ ] **1.3 核心接口开发**
    - [ ] **Auth**: 注册、登录、JWT 中间件。
    - [ ] **User**: 获取用户信息、更新资料。
    - [ ] **Game**: 游戏列表 CRUD (Admin 用)。
    - [ ] **Companion**: 申请陪玩、审核、获取陪玩列表、详情。
    - [ ] **Order**: 下单、接单、确认完成、取消订单、订单列表。
    - [ ] **Wallet**: 充值、余额查询、流水记录。
- [ ] **1.4 接口测试**
    - [ ] 使用 Postman 或 curl 验证所有接口连通性。

## Phase 2: 后台管理开发 (Vue 3 + Element Plus)
- [ ] **2.1 项目初始化**
    - [ ] 在 `workbench` 目录初始化 Vite 项目。
    - [ ] 安装 `element-plus`, `axios`, `pinia`, `vue-router`, `tailwindcss`.
    - [ ] 配置 TailwindCSS 和 Element Plus 样式。
- [ ] **2.2 基础功能**
    - [ ] 登录页面 (管理员登录)。
    - [ ] 布局框架 (Sidebar, Navbar)。
- [ ] **2.3 业务模块**
    - [ ] **用户管理**: 查看用户列表，封禁/解封。
    - [ ] **游戏管理**: 添加、编辑、删除游戏 (上传图片)。
    - [ ] **陪玩审核**: 查看待审核列表，通过/驳回。
    - [ ] **订单管理**: 查看全平台订单状态。
    - [ ] **资金结算**: 查看提现申请 (如有) 或 收益统计。

## Phase 3: 移动端 H5 开发 (UniApp + Vue 3)
- [ ] **3.1 项目配置**
    - [ ] 完善 `h5` 目录结构。
    - [ ] 安装并配置 TailwindCSS (兼容 UniApp 方式)。
    - [ ] 封装 `request.js` (统一处理 Token 和 API URL)。
- [ ] **3.2 公共组件与 UI**
    - [ ] 设计底部导航栏 (TabBar)。
    - [ ] 封装通用卡片、按钮、加载动画组件。
- [ ] **3.3 普通用户功能**
    - [ ] **登录/注册页**: 漂亮的表单 UI。
    - [ ] **首页**: 游戏列表轮播，陪玩推荐列表 (筛选功能)。
    - [ ] **陪玩详情**: 展示资料、价格、下单入口。
    - [ ] **订单流程**: 下单支付模拟 -> 等待接单 -> 服务中 -> 确认完成。
    - [ ] **个人中心**: 订单记录、钱包充值。
- [ ] **3.4 陪玩端功能**
    - [ ] **身份切换**: 个人中心切换到 "接单模式"。
    - [ ] **服务设置**: 编辑负责游戏、价格、在线状态。
    - [ ] **接单大厅**: 轮询新订单，点击接单。
    - [ ] **收益查看**: 查看钱包余额变化。

## Phase 4: 联调与优化
